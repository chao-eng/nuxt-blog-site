## 🎯 设计原则

### 视觉层次
1. **主要内容** - 高对比度，清晰可读
2. **次要内容** - 中等对比度，辅助信息
3. **装饰元素** - 低对比度，视觉引导

### 动画原则
- **时长**: 0.2s - 0.6s（快速响应）
- **缓动**: cubic-bezier(0.4, 0, 0.2, 1)（自然流畅）
- **延迟**: 列表项使用递增延迟（视觉节奏）

### 颜色使用
- **主题色**: #00DC82（绿色）- 强调和交互
- **深空灰**: #09090B - #FAFAFA（背景和文本）
- **透明度**: 0.1 - 0.95（玻璃态效果）

### 间距系统
- **xs**: 0.25rem (4px)
- **sm**: 0.5rem (8px)
- **md**: 0.75rem (12px)
- **lg**: 1rem (16px)
- **xl**: 1.5rem (24px)
- **2xl**: 2rem (32px)

## 🔧 已知问题（可忽略）

### Lint 警告
以下 lint 警告是正常的，不影响功能：

1. **CSS 文件警告** (main.css)
   - `需要冒号`、`应为 {`、`Unknown at rule @theme`
   - **原因**: IDE 不识别 Tailwind CSS 4.0 的 `@import` 和 `@theme` 语法
   - **影响**: 无，运行时正常

2. **line-clamp 警告** (BlogsGridCard.vue)
   - `Also define the standard property 'line-clamp' for compatibility`
   - **原因**: `-webkit-line-clamp` 是标准的多行文本截断方案
   - **影响**: 无，所有现代浏览器都支持

## 📝 使用说明

### 如何启用新导航栏
已自动在 `blog.vue` 布局中启用 `BlogsMainHeaderModern` 组件。

### 如何自定义主题色
编辑 `app/assets/css/main.css` 中的 CSS 变量：
```css
--color-green-400: #00DC82; /* 主题色 */
```

### 如何调整动画速度
修改组件中的 `transition` 时长：
```css
transition: all 0.3s cubic-bezier(0.4, 0, 0.2, 1);
```
