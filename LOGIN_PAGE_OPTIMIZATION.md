## 🎨 动画系统

### 入场动画
```css
/* 装饰内容 */
fadeIn: 0.8s ease-out

/* 表单卡片 */
slideInRight: 0.8s ease-out

/* 特性列表 */
slideInLeft: 0.6s ease-out
延迟: 0.2s, 0.4s, 0.6s
```

### 循环动画
```css
/* 图标浮动 */
float: 3s ease-in-out infinite

/* 装饰圆圈脉冲 */
pulse: 4s, 5s, 6s ease-in-out infinite
```

### 交互动画
```css
/* 输入框焦点 */
transform: translateY(-2px)

/* 按钮悬停 */
transform: translateY(-2px)
box-shadow: 0 10px 20px -5px rgba(99, 102, 241, 0.4)
```

## 📱 响应式设计

### 桌面端 (> 1024px)
- 双栏布局
- 左侧装饰 + 右侧表单
- 1:1 比例

### 平板/移动端 (< 1024px)
- 单栏布局
- 隐藏左侧装饰
- 仅显示表单卡片
- 全屏居中

### 小屏幕 (< 640px)
- 减少卡片内边距
- 2rem → 1.5rem

/* 表单区域 */

- 登录按钮：蓝紫色渐变
